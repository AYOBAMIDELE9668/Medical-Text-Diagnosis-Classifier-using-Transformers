# Medical-Text-Diagnosis-Classifier-using-Transformers
Advanced Healthcare NLP Project (90–93% Accuracy)

This project implements a **medical text diagnosis classifier** that predicts diseases from symptom descriptions using **Transformer-based deep learning models**.

The model is fine-tuned from a pretrained **BioBERT** architecture, enabling it to understand **biomedical language and symptom patterns** more effectively than traditional machine learning models.

The system analyzes symptom text and predicts the most probable disease class.

# Project Overview

Medical Natural Language Processing (NLP) enables machines to understand clinical language and assist in healthcare analytics.
